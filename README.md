# Netflix Content Analysis

## Objective
Analyze Netflix dataset to understand trends, genres, and content distribution.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate Netflix content
- Drama is the most common genre
- USA produces the most content
- Content growth increased after 2015

## Conclusion
This project helped me understand real-world data analysis and visualization.
